Projects

ChaGration

As part of the Aspiring Explorers in Science (AXiS) research program hosted by Tencent, I collaborated with a partner to develop ChaGration, a tool designed for migrating software tests across different mobile applications. Under the guidance of Professor Tao Xie and Ph.D. student Dezhi Ran from Peking University, we explored the use of conversational large language models to compare semantic similarity between various user interface (UI) elements through prompt engineering. This novel approach enabled our system to reach a test migration accuracy of 90% with an average runtime of just 3 minutes—far surpassing the efficiency and effectiveness of previous methods. The project incorporated tools such as OpenAI’s API client, Android Emulator, Appium 2.0, ADB, and Python. The result is reported in here.

LeSort: An Automated Lego Sorting Machine

In early 2025, I designed and built LeSort, an automated Lego sorting machine capable of categorizing pieces by size, color, or type. The system integrated an online computer vision and machine learning (CVML) model that allowed users to adjust sorting strategies according to their needs. A user-friendly GUI provided real-time feedback and control during the sorting process. The machine achieved a 90% sorting accuracy with an average operation time of 2.5 seconds per piece, drastically reducing the time and effort required for manual sorting. This project utilized technologies such as OpenCV2, TKinter, Threading, Onshape, and Arduino.
